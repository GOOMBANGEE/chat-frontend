# [Chat.goomba](https://chat.goomba.org)
STOMP를 이용하여 실시간으로 개인, 그룹채팅을 제공합니다.  
텍스트및 이미지 업로드가 가능합니다.  

Created with [React](https://react.dev/) + [Tailwind CSS](https://tailwindcss.com/)

## Environment
Typescript  
React 18.3.1  
Node.js 20.14

axios  
react-cookie  
react-router-dom@6  
tailwindcss  
use-sound  
vite  
ws  
zustand  

## Development

```
git clone https://github.com/GOOMBANGEE/chat-frontend.git

cd chat-frontend
cp .env.example .env

npm install

npm run dev
```
